# AGENTS Strategy

## The Mandate
Create a live personal portfolio website that positions you as a junior **Frontend Developer with UX focus** targeting **defense tech and pharmaceutical companies in New Jersey**. The goal is to get your first entry-level role in enterprise UI, internal dashboards, or healthcare workflow interfaces.

## The Tech Stack
- **HTML / CSS / JavaScript**
  - Why: Core frontend skills that prove you can build accessible, responsive UI and implement polished interactions.
- **Figma**
  - Why: Shows you can design interfaces, create visual layouts, and communicate UX decisions.
- **Accessibility best practices**
  - Why: Differentiates you for regulated industries and improves usability for enterprise/internal tools.
- **SEO-friendly copy**
  - Why: Helps the portfolio rank for NJ/NY job searches and shows attention to digital presence.

## The Sitemap
- **Hero**
  - Clear headline: frontend developer with UX/design focus for NJ defense and pharma.
  - Short value proposition, local relevance, and quick proof points.
- **Case Studies**
  - Use existing class projects as framed portfolio entries.
  - Include project goals, your role, tech used, design/UX impact, and lessons learned.
  - Add a self-case-study for the portfolio build itself.
- **Lead-Gen Form**
  - Simple contact form with name, email, role interest, and message.
  - Include a strong CTA and option to request a portfolio review or entry-level frontend role conversation.

## Non-Negotiables
- **Mobile-responsive**
  - The site must work well on phones and tablets, with a clean responsive layout and readable text.
- **SEO-optimized keywords for NJ/NY**
  - Include phrases like: `New Jersey frontend developer`, `NJ UX/UI designer`, `defense tech UX`, `pharma UI developer`, `entry-level frontend NJ`, `NJIT web developer`.
- **Specific Call to Action**
  - Example CTA: `Let’s build UX-driven frontend tools for NJ defense and pharma teams.`
  - End with a clear form prompt: `Contact me to discuss junior frontend or UX/UI roles in New Jersey.`
